# Miscellaneous Commands

## Birthday

{% hint style="danger" %}
This command requires you to have the **MANAGE\_MESSAGES** permission.
{% endhint %}

### What this command does

This command sends an embed wishing the mentioned user a happy birthday.

### How to use it

Simply type `c;birthday [@user]` and a message will be sent out!

## Verify

{% hint style="info" %}
This command is only used in the \#verification channel. Chances are, if you can't see it, you're already verified.
{% endhint %}

### What this command does

It allows you to gain access to our server.

### How to use it

Type `c;verify <your Minecraft IGN>` and simply wait.
