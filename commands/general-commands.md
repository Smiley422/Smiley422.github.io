# General Commands

{% hint style="info" %}
Please take note of the following: arguments that are surrounded by "\[" and "\]" are required. Arguments that are surrounded by "\(" and "\)" are optional.
{% endhint %}

## About

### What this command does

The about command lists information about Condemned Moderation

### How to use it

Simply type `c;about` in any channel.

## Bot info

### What this command does

This command lists information about the bot, such as the current Node version that is running, the Discord.js version it is running, the bot's version, and the RAM it is using.

### How to use it

Simply type `c;botinfo` and you're all set!

## Help

{% hint style="info" %}
You must allow DMs from bots for this command to work.
{% endhint %}

### What this command does

The command sends the user a DM with all of the commands.

### How to use it

Type `c;help`.

## Mojang status

### What this command does

This command lists the status of the Mojang API.

### How to use it

Simply type `c;mojang` and you're all set!

## Server info

### What this command does

This command lists information about the server you're in.

### How to use it

Simply type `c;serverinfo` and you're good to go!

## User info

### What this command does

This command allows you to view information about a user.

### How to use it

Simply type `c;userinfo [@user]` and you're good to go!
