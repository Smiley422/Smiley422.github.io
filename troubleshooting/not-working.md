# What to do if the bot isn't responding

First things first, please make sure that the bot is online. If the bot has the status of online, idle, or do not disturb, the bot is capable of receiving requests and handling commands.

{% hint style="warning" %}
While certain staff in our Discord may be able to fix parts of the bot, it is recommended that you tell Smiley after trying all of the things listed below.
{% endhint %}

* Verify that Condemned Moderation has the following roles:
  * Management
  * Automation
  * Bots
  * Verifier
  * Member
* Make sure that the role **Management** is able to do the following:
  * Send messages
  * Read message history
  * Manage messages
  * Send external emojis
  * Manage roles
  * Manage channels
  * Kick members
  * Ban members
  * Embed links
  * Add reactions
* Verify that Condemned Moderation responds to commands
  * Type `c;ping` in any channel and see if the bot responds
* Verify that the bot's status is set to something. The status of the bot should _never_ be blank.

If you've tried all of these, and are certain you've checked all of these, please DM Smiley about your issue. \(Smiley422\_2004\#9634\)
